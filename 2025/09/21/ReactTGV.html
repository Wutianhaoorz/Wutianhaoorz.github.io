TGV-R (3D reacting Taylor–Green vortex)

Physical description

We consider a periodic cubic domain $\Omega=[0,L]^3$ discretized on a structured grid of $256\times256\times256$ cells, with side length $L=2\pi\,\mathrm{mm}$ and uniform spacing $\Delta x=L/256$. The fundamental wavenumber is $k=2\pi/L$. The initial velocity follows the Taylor–Green form:

[ u(\boldsymbol{x},0)=U_0\sin(kx)\cos(ky)\cos(kz), ]

[ v(\boldsymbol{x},0)=-U_0\cos(kx)\sin(ky)\cos(kz), ]

[ w(\boldsymbol{x},0)=0, ]

with Reynolds number $\mathrm{Re}=U_0 L/\nu(T_0)$ based on the kinematic viscosity $\nu$ at the reference temperature $T_0=300\,\mathrm{K}$. The composition is non-premixed and blended by a smooth mask:

[ \psi(\boldsymbol{x})=\tfrac12\Big[1+\tanh!\Big(c\,\frac{R_d(\boldsymbol{x})-R}{R}\Big)\Big], ]

where $R_d(\boldsymbol{x})$ is the distance to the prescribed mixing centerline or surface, $R$ sets the characteristic half-width, and $c$ controls interface sharpness (here fixed as $c=3$). Following the reference setup, we prescribe side values $Y_{\mathrm{H_2}}^{0}=0.0556$ on the fuel side and $Y_{\mathrm{O_2}}^{0}=0.233$ on the oxidizer side, with nitrogen complement elsewhere. The initial species fields are:

[ Y_{\mathrm{H_2}}(\boldsymbol{x},0)=Y_{\mathrm{H_2}}^{0}\,[1-\psi(\boldsymbol{x})], ]

[ Y_{\mathrm{O_2}}(\boldsymbol{x},0)=Y_{\mathrm{O_2}}^{0}\,\psi(\boldsymbol{x}), ]

[ Y_{\mathrm{N_2}}(\boldsymbol{x},0)=1-Y_{\mathrm{H_2}}(\boldsymbol{x},0)-Y_{\mathrm{O_2}}(\boldsymbol{x},0), ]

with all other species initially zero, and the pressure uniform at $p_0$. After blending, each grid point is brought to a thermochemical equilibrium state by solving a local equilibrium at $T_0=300\,\mathrm{K}$ and $p_0$, yielding the final initial temperature and species fields used to start the simulation.

Simulation method

Solutions are computed with the DeepFlame-based implicit DNS setup on the native mesh. The time step is fixed at $\Delta t=10^{-6}\,\mathrm{s}$. Snapshots are written every 10 time steps (that is, every $10\Delta t$), and each trajectory contains 40 uniformly spaced frames on the native mesh. The reaction mechanism is a reduced H$_2$/air scheme with nine species and twelve reversible reactions.

Varied physical parameters

We generate 16 trajectories by crossing two knobs that set the relative turbulence and mixing scales: $U_0\in{4,8,16,32}\,\mathrm{m/s}$, corresponding to $Re\approx{1.7\times10^3,\,3.3\times10^3,\,6.7\times10^3,\,1.3\times10^4}$, and the initial mixing half width $R\in{\tfrac{\pi}{4},\,\tfrac{3\pi}{8},\,\tfrac{\pi}{2},\,\tfrac{5\pi}{8}}\,\mathrm{mm}$. Increasing $U_0$ shortens the characteristic time scale and intensifies turbulence, while varying $R$ changes the initial layer thickness.

Provided data

Each snapshot includes temperature $T$, pressure $p$, density $\rho$, velocity components $(u,v,w)$, and nine species mass fractions $Y_k$ (reduced $\mathrm{H_2}$–air mechanism). Data are provided on the native $256\times256\times256$ mesh together with spatial coordinates and a fixed train/validation/test split by trajectory. For memory-constrained experiments, we also release an $8\times$ downsampled version in each spatial direction.
